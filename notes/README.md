# Notes

This directory contains dated working notes and planning documents that are useful
for project history but are not part of the core analysis/workflow documentation.

## Structure

- `manuscript-docx/`: manuscript prep checklists and number verification notes
- `plans/`: dated planning/design notes
- `worklog/`: dated session summaries and progress notes

## Guidance

- Keep durable technical documentation in `docs/`.
- Place time-bound, draft, or operational notes in `notes/`.
- Prefer date-prefixed filenames (`YYYY-MM-DD-...`) for new entries.
